# DNA Analyzer V1

## Description

DNA Analyzer V1 is a simple Python program that validates a DNA sequence and performs basic analysis. It calculates the sequence length, counts each nucleotide (A, T, G, and C), and determines the GC and AT percentages.

---

## Features

- Validate DNA sequences (A, T, G, C only)
- Calculate DNA sequence length
- Count the number of A, T, G, and C
- Calculate GC percentage
- Calculate AT percentage
- Detect and reject invalid DNA sequences

---

## Example

### Input

```text
Enter The DNA Sequence:
ATGCGATTA
```

### Output

```text
GC Percentage: 33.33
AT Percentage: 66.67

Count of A: 3
Count of T: 3
Count of G: 2
Count of C: 1

Length Of DNA Sequence: 9
```
